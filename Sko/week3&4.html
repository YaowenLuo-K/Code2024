Class activity Make paper-prototype(Pacman game) Process This is our group's paper prototype inspired by the Pacman game.Before creating the prototype, we studied the game's underlying logic and the interactions required by users. Through our case study, we identified the core elements that make up the game: the map, character, control buttons, and coins. We made the coins using paper strips, threading each through both ends of the map. Pulling the strips simulates the character collecting coins as they move, creating a dynamic, hands-on experience of the game mechanics.
Reflection This class activity helped me build my own paper prototype for Assignment 1. I learned that creating a paper prototype is valuable for visualizing and structuring the basic elements of a project before coding. By mapping out interactions and the logic visually, I could identify essential functions and better understand how the program should flow. This hands-on approach will make the coding process smoother and more organized, helping me translate ideas into working code more effectively.
